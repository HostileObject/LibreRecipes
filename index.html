LibreRecipes

LibreRecipes is a HTML-and-CSS-only, JavaScript-free project inspired by Luke Smith. The aim of this project is to offer a fast and bloat free database of recipes, without any JavaScript file loading in the background and without any useless text from the recipes author. If you would like to contribute contact me at samueletonarini@gmail.com or visit the GitHub page.
